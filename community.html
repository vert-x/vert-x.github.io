Project and Community Information

The Project

Vert.x is an independent project (not owned or controlled by any particular company), and includes contributions from many individuals and organisations.

The core Vert.x project will shortly be a project of the Eclipse Foundation The Eclipse project proposal provides some useful background information including a list of interested companies and individuals.

The Vert.x source can be accessed at github where the vert-x organisation is the umbrella under which all the Vert.x related projects live. This includes the core Vert.x project, as well as various others such as language implementation modules, the test framework, web-site and others.

Vert.x is highly modular and the core project is quite compact. Much of the functionality of Vert.x is provided in the form of modules - some maintained under the vert-x umbrella but many others created and maintained by the community.
